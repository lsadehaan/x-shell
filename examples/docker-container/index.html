x-shell Docker Container Example

How to Test

1. Make sure Docker is running on this machine.
2. Start a test container: docker run -d --name test-container alpine sleep infinity
3. The terminal will automatically detect Docker is enabled and show available containers.
4. Select a container from the dropdown and click "Connect".
5. You'll get a shell inside the container!
6. Try running: cat /etc/os-release to see you're inside Alpine Linux.
7. Use the settings dropdown (gear icon) to change theme and font size.
