x-shell Session Multiplexing

Share terminal sessions between multiple browser tabs, with tabbed terminals

How to Test Multiplexing

1. Select New Local Shell and click Start Session
2. Run some commands to generate output
3. Open this page in a second browser tab
4. In tab 2, select Join Existing Session from the Mode dropdown
5. Select the session and click Join Session
6. Both tabs now share the same terminal!

Tip: Close tab 1 - the session survives! Tab 2 can continue using it.
